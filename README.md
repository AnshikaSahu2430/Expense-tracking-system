# 💰 Expense Tracking System

A full-stack MERN application that helps users manage their personal finances by tracking income and expenses, analyzing spending patterns, and generating financial reports.

## 🚀 Features

### 🔐 Authentication
- User Registration
- User Login
- JWT Based Authentication
- Protected Routes
- Secure Password Hashing using bcrypt

### 💵 Income Management
- Add Income Sources
- View Income History
- Delete Income Records

### 💸 Expense Management
- Add Expenses
- Categorize Expenses
- View Expense History
- Delete Expense Records

### 📊 Dashboard & Analytics
- Total Income Calculation
- Total Expense Calculation
- Remaining Balance
- Monthly Financial Summary
- Expense Analysis Charts

### 📄 Reports
- Generate Financial Reports
- Export Reports as PDF
- Monthly Expense Tracking

## 🏗️ Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt.js
- Chart.js
- Axios
